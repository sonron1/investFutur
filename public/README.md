# InvestFuture - Plateforme d'investissement participatif

## 📋 Description

InvestFuture est une plateforme d'investissement participatif dédiée à l'innovation durable. Elle permet aux investisseurs d'accéder à des opportunités d'investissement exceptionnelles dans des entreprises à fort impact environnemental, avec un ROI moyen de 18%.

## ✨ Fonctionnalités principales

- **Investissement minimum** : À partir de 5 000€
- **ROI moyen** : 18% annuel
- **Dividendes réguliers** pour les investisseurs
- **Projets présélectionnés** par des experts
- **Transparence totale** sur les investissements
- **Plateforme régulée** et sécurisée (FSMA, AMF, ACPR)
- **+2 500 investisseurs actifs**

## 🎯 Secteurs d'investissement

- Innovation durable
- Technologies vertes
- Entreprises à impact environnemental positif
- Projets d'avenir

## 🛠️ Technologies utilisées

- **Frontend** : HTML5, CSS3, JavaScript
- **Framework CSS** : Bootstrap 5.3.0
- **Icônes** : Font Awesome 6.4.0
- **Typographie** : DM Sans (Google Fonts)
- **Animations** : AOS (Animate On Scroll)
- **Design** : Responsive et moderne

## 🚀 Installation & Utilisation

1. **Cloner le projet**
   ```bash
   git clone [url-du-repo]
   cd investissements
   ```

2. **Lancer le projet**
   - Ouvrir `index.html` dans un navigateur web moderne
   - Ou utiliser un serveur web local (WAMP, XAMPP, Live Server, etc.)

3. **Développement local**
   ```bash
   # Avec Python
   python -m http.server 8000
   
   # Avec Node.js (http-server)
   npx http-server
   ```
